# School 21 Slides Site

Отдельный статический сайт-сосуд для подготовки и демонстрации презентационных материалов для Школы 21.

- `index.html` — новая самостоятельная страница для проекта Школы 21.
- `style.css` — официальный сине-бирюзовый визуальный язык по переданному референсу.

Публикация: GitHub Pages из ветки `main`, папка `/`.
